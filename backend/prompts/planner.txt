Given the ICP configuration and business context below, create a detailed execution strategy that tells
the AI agent pipeline exactly what to do, in what order, and with what priority.

The execution strategy must specify:
1. Which agents to run (from the available set)
2. Whether to run them sequentially or in parallel
3. The priority order for each agent
4. The LLM model to use for each agent (gemini-2.0-flash-exp for speed, gemini-1.5-flash for quality)
5. The maximum number of companies to discover and process
6. Expected output targets

Available agents:
- trigger_monitoring: Scans news/RSS for buying trigger signals
- company_discovery: Finds companies matching ICP from web sources
- company_validation: Validates company data against ICP rules
- company_enrichment: Deep enrichment with tech stack, hiring, news, funding
- contact_discovery: Finds key decision-maker contacts at validated companies
- next_best_action: Determines optimal outreach action for each company
- business_brief: Generates comprehensive sales intelligence brief

{icp_config}

Business Context:
{business_context}

    }
  ],
  "targets": {
